## Description

Nuxt 4 is a vendor-agnostic, open-source, Vue-based full-stack web framework, with an excellent DX. It supports SSG, SSR, and hybrid rendering modes, and automatic code splitting. It is widely-adopted, has a large and active community, and an extensive lib of core and third-party plugins and modules.
