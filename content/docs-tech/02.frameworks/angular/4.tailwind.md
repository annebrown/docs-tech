## Install

Install `tailwindcss` and `postcss` as dev deps:

```bash
pnpm add -D tailwindcss postcss autoprefixer
```

## Config

Configure Tailwind to scan for implemented class names:

```ts
// tailwind.config.js
module.exports = {
    content: [
        './src/**/*.{html,ts}',
        './src/app/**/*.{html,ts}',
        './src/components/**/*.{html,ts}',
    ],
    theme: {
        extend: {},
    },
    plugins: [],
}
```

Add Tailwind Directives to styles:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
