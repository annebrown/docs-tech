## Dev Server

Configure app with local IP addy and assign an open port number:

`angular.json`:

```json
{
    "projects": {
        "angular-base": {
            "architect": {
                "serve": {
                    "options": {
                        "browserTarget": "your-project-name:build",
                        "host": "192.168.22.222" // Dev Svr IP
                        "port": 3002 // Dev Svr Port
                    },
      ...
```
